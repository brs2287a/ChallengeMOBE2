# MiniProjetMOBE, membres : Esteban BRAVO, Edouard LEBEAU, Emma MAUGER

Pensez à sync Gradle sinon firebase ne peut pas fonctionner

Règles et fonctionnalités du jeu :

- éviter de toucher les bords
- incliner la tablette dans la direction où va la balle l'accélère (l'inverse la ralenti, grâce à l'accéléromètre)
- toucher l'écran pour que la balle change de direction (attention elle accélère à chaque fois, touch screen)
- se mettre à l'ombre peut être dangereux ;) (changemnt de couleur avec le capteur de luminosité)
- affichage des meilleurs scores locaux (sharedPreference) et mondiaux avec Firebase
- page d'accueil avec possibilité d'inscrire un pseudo
